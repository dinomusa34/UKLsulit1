Cara Kerja Program
Program ini menghitung rata-rata nilai dari sejumlah siswa berdasarkan input pengguna.
Langkah-langkah:
- Pengguna memasukkan jumlah siswa.
- Program memvalidasi bahwa input adalah bilangan bulat positif.
- Untuk setiap siswa, pengguna diminta memasukkan nilai:
- Jika input bukan angka, program meminta ulang.
- Jika nilai negatif, dianggap 0.
- Semua nilai dijumlahkan, lalu dihitung rata-ratanya.
- Hasil rata-rata ditampilkan dengan dua angka di belakang koma.
